## 주요 제약/주의사항

1. **네이버 광고 콘솔 전환매출은 부풀려져 있음** — "총 전환매출" 컬럼은 장바구니 가치 포함, 실매출의 약 5배. 실_ROAS는 `구매전환매출` 컬럼 사용.
2. **CS 경유 오프라인 전환 비중 큼** — 사인물 업종은 CS/전화/카톡 주문 많음. 온라인 전환 0건이어도 매출 0이 아님. 소재 OFF/입찰 인하는 CS 확인 후 판단.
3. **쇼핑검색광고 API PUT 요구사항** — `type`/`nccAdId` 필수, `bidAmt`는 10원 단위, `adAttr.bidAmt` 경로, `?fields=adAttr` 쿼리.
4. **n8n은 JSON import 방식** — 브라우저 조작 금지, JSON export→수정→import (10배 빠름).
