**Why**: 2026-04-24 차장님이 리포트 채널별 광고 출처를 정확히 정리해줌. 지금까지 리포트에는 스마트스토어 외 채널이 "광고 무관"으로 표시되었지만 실제로는 각 채널마다 광고 매체가 존재. 네이버 광고만 파이프라인에 연결되어 있고 나머지는 수동 관리 중.

**How to apply**:
- 리포트 UI 업데이트: 각 채널 아래 광고매체 이름 + 연동상태 뱃지 표기
  - 예: `자사몰(hanasignmall.com)  ▸ 네이버 B계정  ⏳ 연동중`
- 우선순위 재정립:
  - 1순위 A/B: 네이버 B계정 수집+Conv 복제 → 자사몰.com ROAS 표기
  - 2순위 C: 스케줄 09:30 + ADVoost 자동 수집
  - 신규 K/L/M: 구글애즈·쿠팡·ESM 순차 연동 (각 1~2일 예상)
  - 보류 H: 실결제 대조 시스템 (Conv API 재검증 결과 필요성 하락)
- 각 연동은 독립적이므로 개발 병렬 가능. 단 리포트 통합 표시는 마지막에 한꺼번에 UI 업데이트
- 업체계약현황 시트의 진행자 명칭이 광고 매출 추적의 기준점 — 새 진행자 추가 시 이 매핑에도 업데이트 필수
