**규칙**: 하나사인몰 광고 성과 평가 시 "네이버 전환 0건" 을 "매출 0원"으로 단정하지 말 것. 특히 소재 OFF, 입찰가 인하 같은 되돌리기 어려운 조치는 온라인 전환 데이터만으로 결정 금지.

**Why**: 하나사인몰은 게시판/안내판/입간판/주차스티커/바닥스티커/현수막 등 사인물 제작업이다. 아파트/학교/공공기관/시설물 단체 주문이 많고, 고객들이 규격·재질·시공·수량·견적 상담을 위해 **전화, 카카오톡, 메일로 직접 문의**하는 경우가 매우 많다. 광고 클릭해서 상세페이지 보고 → 전화로 문의 → 견적 후 주문 경로가 일반적. 네이버 광고 전환 추적은 온라인 구매완료만 잡아내므로, CS 경유 전환은 완전히 보이지 않음. 박상준 차장이 2026-04-14 에 "CS 연락 많으니 함부로 소재 입찰가 수정하면 위험"이라고 명시.

**How to apply**:
- 광고 성과 판단 시 "네이버 구매전환수 = 0" 만 보고 OFF/인하 권고 금지
- 반드시 고려할 항목: CS 문의량, 전화 주문량, 매출 트렌드, 시즌성, 해당 소재의 키워드/상품 특성
- 권장 절차: 광고 성과가 의심되는 소재 발견 → **CS 팀에 해당 상품/키워드 문의 유입 확인** → 오프라인 전환 연결 가능성 체크 → 그 후 판단
- 절감 제안 시 "입찰가 조정" 보다 "상세페이지 개선", "CS 응대 구조 개선", "콜트래킹 도입" 같은 간접 개선 우선 제안
- 장기 해결책: 콜트래킹(CallRail 등) 도입으로 광고 → CS 전환 추적, 또는 CS 응대 시 "어떤 광고 보고 오셨나요" 질문으로 수집
- 하나사인몰 관련 마케팅 조언 시 항상 "온라인 전환 + 오프라인 CS 전환" 두 축 모두 고려할 것
