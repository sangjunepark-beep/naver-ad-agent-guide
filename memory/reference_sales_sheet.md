## 일자 컬럼 운영 규칙 (2026-04-24 확정)

- 엑셀 **"D일자" 행 = 실제 주문 [D-1 13:00 ~ D 13:00]**
- 오전 9시 취합: 전날 13시 이후 주문 → 당일 일자로 기입
- 오후 1시 취합: 당일 9시~13시 주문 → 당일 일자로 기입
- 이후(D 13시 이후) 주문은 **다음날 일자**로 기입
- 따라서 **엑셀 D일자 ≠ 네이버 API D일자(KST 자정 기준)** — 단일 일자 1:1 대조 불가, 누적 기간 대조만 유의미
