**사실**: 2026-04-14 확인 — 네이버 검색광고 "총 전환매출액"은 장바구니 담기(add_to_cart) 이벤트 가치까지 합산. 하나사인몰 04-13 기준 콘솔 8,210,490원 vs 실제 결제 1,644,300원 (5배 부풀림).

**Why**: 네이버 + 스마트스토어 연동 기본 설정에서 `add_to_cart` 이벤트에도 상품 가격이 value로 전송됨. 네이버 "총 전환매출"은 모든 convType의 value 합산이라 장바구니만 담고 안 산 건도 매출로 찍힘. 대표 사례: nad-a001-02-000000282128922 — 실제 구매 1건 56만원 + 장바구니 6건 580만원 = 콘솔 638만원. 이 한 소재가 전체 04-13 콘솔 매출의 77% 차지.

**How to apply**:
- 매출/ROAS 보고 시 반드시 `02_전환수집로그` 시트의 `구매전환매출` 컬럼 사용 (purchase만)
- `장바구니추정매출`은 장바구니 이탈 분석 시에만 참고, 매출 합산 금지
- 네이버 콘솔 대조 필요 시 `구매전환매출 + 장바구니추정매출`로 계산
- Phase 3 대시보드는 실_구매전환매출 기준
- 광고비 낭비 현황: 04-13 기준 177개 집행 소재 중 162개(91.5%)가 전환 0건, 낭비 비용 484,692원/일 (월 환산 약 1,450만원)
- 자동입찰 모드는 허수 지표 기반이므로 수동 입찰 검토 권장
- 대행사 리포트도 대부분 이 허수 포함이므로 반드시 자체 데이터로 교차검증
